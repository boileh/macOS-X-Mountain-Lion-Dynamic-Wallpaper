# How to use:
1: Download the Latest Release

2: Open Settings

3: Find Wallpapers

4: Click Add Photo

5: Click "Choose"

6: Find OS X Mountain Lion Dynamic.heic

7: Use as Wallpaper

8: Set the Wallpaper to Dynamic
